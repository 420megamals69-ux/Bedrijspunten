Programming Foundations: Fundamentals

Programmeer basiskennis

Wat is programmeren?

Programmeren is het omzetten van ideeën tot specifieke en sequentiële instructies die een computer kan begrijpen en uitvoeren. Onjuiste instructies en een onjuiste volgorde van stappen kunnen leiden tot bugs of crashes. Een bug is een onverwachts gebeurtenis en een crash verwijst naar het voortijdig stoppen of vastlopen van een programma.

Wat is een programmeertaal?

Een programmeertaal wordt gebruikt om een computer instructies te geven. Programmeertalen zijn high-level languages: er wordt gebruik gemaakt van sleutelwoorden, structuur en syntax, zodat de code makkelijker te begrijpen is voor mensen. De ingevoerde instructies worden vervolgens omgezet tot de taal die computers begrijpen: Machine language.

Elke taal heeft sterke en zwakke punten: Sommige zijn ideaal voor het programmeren van kleine apparaten met een beperkt geheugen, terwijl andere gemaakt zijn om complexe wiskundige berekeningen uit te voeren. Daarnaast heeft elke programmeertaal een eigen syntax:

Source code schrijven

Code is de instructie die voor de computer worden geschreven in plain tekst: tekst zonder speciale opmaak, zoals vet, cursief of verschillende lettertypes.

Tekstverwerkingsprogramma's voegen standaard stukjes informatie toe in bestanden en zijn niet geschikt voor het schrijven van code. Teksteditors zoals Notepad worden gebruikt.

Elke programmeertaal heeft eigen bestandsextensies. Voor JavaScript is dat bijvoorbeeld .js, voor Perl .pl en voor Kotlin .kt.

Interpreters verwerken source code elke keer dat deze wordt uitgevoerd, regel voor regel. De andere gebruikers moeten een interpreter beschikbaar hebben om hier gebruik van te maken en er zijn extra stappen nodig om de code te runnen.

Een IDE gebruiken

Integrated development environments zijn teksteditors met functies die de ontwikkeling van code versnellen. Denk aan hulpmiddelen voor het schrijven, debuggen en compileren van code. Voorbeelden zijn Xcode en Visualstudio.

Programmeer syntax

Python runnen op de command line van de MAC

Code kan worden uitgevoerd vanuit het command line-terminal. Een MAC-gebruiker moet eerst "python3 + enter" in het terminal invoeren, voordat python instructies kunnen worden ingevoerd. De computer geeft aan dat het gereed is voor python met >>>. In het geval van Windows is het Python-terminal direct gereed voor het uitvoeren van Pyhon instructies.

Syntax refereert naar de regels van een programmeertaal. Syntax errors vinden plaats als de geschreven code de regels van de programmeertaal breekt, bijvoorbeeld als een uitdrukking incompleet is.

Change directory, oftewel "cd", gevolgd door de file locatie, maakt het mogelijk om naar de directory van een file te navigeren. In Windows is het mogelijk om de file path te selecteren door eerst "cd" te typen en vervolgens de gewenste file in het CMD-terminal te draggen.

Doormiddel van "dir+ enter" worden de files in de directory zichtbaar. Mac en Linux gebruiken "ls + enter".

Om een file te kunnen runnen in de terminal van een MAC moet wederom "python3" worden getypt, gevolgd door de file naam. Voor het Windows CMD-terminal is "python" gevolgd door de file naam genoeg.

Simpele uitdrukkingen en expressies

Uitdrukkingen zijn de bouwstenen van elk programma en zijn de individuele acties die het programma moet ondernemen. Uitdrukkingen die een wiskundige berekening kunnen uitvoeren, iets op het scherm kunnen weergeven of een keuze kunnen maken tussen twee codepaden. Elke uitdrukking kan bestaan uit sleutelwoorden, uitdrukkingen en operatoren.

Operatoren zijn symbolen die de computer vertellen om een actie uit te voeren met bepaalde invoer. Een expressie is een combinatie van operatoren en inputs die samen één waarde vormen. De tweede vorm van runnen start tevens een Python-shell op, waarin andere Python instructies kunnen worden ingevoerd. Net als de CMD-terminal wordt dit afgesloten met exit().

Operatoren Input/operand Expressie
+ - * / 2 + 2 2 + 2 = 4

De hiërarchie die de operatoren hebben, is tijdens het programmeren hetzelfde als bij rekenen: Eest haakjes, dan exponenten, vervolgens vermenigvuldigen en delen, en als laatst optellen en aftrekken.

Variabelen en datatypes

Introductie tot variabelen en datatypes

Een variabel is een houder van waarde die het mogelijk maakt om later naar iets terug te refereren. Variabelen bestaan uit een toegewezen naam en waarde die aan elkaar worden toegewezen met een "=". Twee voorbeelden hiervan zijn naam = "Vincent" en straatnummer = 52.

Bij de eerste waarde worden er dubbele aanhalingstekens gebruikt en bij de tweede geen, omdat de datatypes variëren. Met een datatypes kunnen variabele in een bepaalde categorie worden geplaatst, zodat de computer weet hoeveel geheugenruimte nodig is. De eerste variabel is een string en de tweede een integer.

De datatype kan worden achterhaald met de functie "type", bijvoorbeeld print(type(naam)) of print(type(straatnaam)).

Variabelen in verschillende talen

Variabelen kunnen op verschillende manieren reageren op basis van de programmeertaal. Bij striktere programmeertalen moeten de datatypes van de variabelen voor het gebruik worden gedefinieerd, zoals bij Java, C# en C++. Het veranderen van deze variabel is achteraf ook niet mogelijk bij een striktere taal. Flexibelere talen vereisen over het algemeen minder code en striktere talen zorgen voor minder onverwachte verassingen.

Er zijn verschillende regels tijdens het werken met variabelen in Python:

Regel
Een variabele mag alleen letters, cijfers en underscores bevatten.
De naam van een variabel mag niet beginnen met een cijfer.
Namen van variabelen zijn hoofdletter gevoelig.
Variabelen kunnen geen sleutelwoorden zijn.
De naam moet een korte en duidelijk te begrijpen omschrijvingen zijn.

Door "import keyword" te combineren met "print(keyword.kwlist)", kan de complete lijst van sleutelwoorden worden opgeroepen in Python.

Bij grote getallen zoals 45.000 moet het gebruik van komma's worden vermeden, aangezien dit tot semantische fouten leidt.

Werken met strings

Een string is een reeks tekens achter elkaar en kan bestaan uit letters, cijfers, symbolen en spaties. Een string wordt meestal uitgedrukt met aanhalingstekens.

In Python kunnen variabelen met enkele en dubbele aanhalingstekens worden uitgedrukt, zolang beidde tekens met elkaar overeenkomen. Dit maakt het volgende mogelijk om de volgende functies uit te voeren zonder syntax errors: print('She said "Unbelievable" to him.') en print("Let's get started").

Correct witruimtes gebruiken

Het is gebruikelijk om extra spaties en lege regels aan code toe te voegen om de code makkelijk leesbaar te maken. Het verwijderen van witruimtes tussen sleutelwoorden leidt tot syntax errors.

Werken met comments

Comments zijn kanttekeningen waarmee wordt beschreven wat de code doet. Het doel van de verschillende variabelen, functies en klassen kan worden verduidelijkt doormiddel van comments. Comment worden aangeroepen met een "#" en worden over het algemeen boven of achter een bestaand stuk code worden geplaatst. Comments kunnen ook gebruikt worden om tijdelijk stukken code te negeren. Dit heet commenting out code en is mogelijk omdat comments geen invloed hebben op de output van de code.

Voorwaardelijke code

Beslissingen maken in code

Een conditinal of boolean uitdrukking verwijst naar een uitdrukking die tot een waarde van true of false leidt. De vraag "is 4 gelijk aan 2*2" is hier een voorbeeld van. Relationele operatoren worden gebruikt, om dit soort vragen aan de computer te stellen. Relationele operatoren maken gebruik van twee operanten en geven een waarde van true of false op basis van de onderlinge relatie.

De "gelijk aan" operator is een relationele operator en wordt gerepresenteerd met "==". Bijvoorbeeld "4 == 2*2 => True" of "naam = "Vincent" naam == "Tijmen" => False". Een veelvoorkomende fout is het gebruik van "=" in plaats van "==" tijdens het checken ven gelijkheid.

De volledige lijst van relationele operatoren:

Symbool Operator Voorbeeld
== Gelijk aan 3 == 3 // True
!= Niet gelijk aan 3 != 3 // False
< Kleiner dan 3 < 4 // True
> Groter dan 3 > 4 // False
<= Kleiner of gelijk aan 3 <= 4 // True
>= Groter of gelijk aan 3 >= 3 // True

Conditionele code verkennen

De if uitdrukking is de meest voorkomende manier waarop beslissingen worden gemaakt met de computer. If uitdrukkingen beginnen met het woord if, gevolgd door een conditie en een dubbele punt. Een handeling wordt uitgevoerd als het antwoord true is. Denk bijvoorbeeld aan: 

if 5 < 6: 
    print("Ja, 5 is kleiner dan 6.")
    print("Iedereen weet dat!")

De insprong is belangrijk aangezien Python hiermee identificeert welke code tot de if uitdrukking behoord. Uitdrukkingen die op deze manier zijn gegroepeerd, heten een block. Het block eindigt bij de eerstvolgende lijn die niet is ingesprongen. Het bovenstaande block bevat twee uitdrukkingen.

Werken met simpele voorwaardelijke uitdrukkingen

Met de if-else uitdrukking wordt er ook een handeling uitgevoerd als het antwoord false is. Een else wordt nooit op zichzelf gebruikt; Het volgt altijd een eerdere if uitdrukking:

if 5 > 6: 
    print("5 Is groter dan 6."Vreemd!)
else:
    print("Ja, 5 is kleiner dan 6.")

Condities in verschillende talen

De if-else uitdrukking heeft een verschillende syntax op basis van de programmeertaal. Dit is de syntax in Java:

if (5 > 6) {
    System.out.println("5 Is groter dan 6."Vreemd!);
} else {
    System.out.println ("Ja, 5 is kleiner dan 6.");
}

In Java is het gebruikelijk om de uitdrukkingen in het block in te laten springen, maar dit is niet noodzakelijk. Hetzelfde geld voor een programmeertaal als Ruby:

if 5 > 6
    puts "5 Is groter dan 6."Vreemd!
else
    puts "Ja, 5 is kleiner dan 6."

end

Modulaire code

Introductie tot functies

Een functie is een block dat omsloten is met een naam. Functies maken het lezen, schrijven en debuggen van programma's makkelijker. Voorbeelden hiervan zijn print() en input().

Creëren en roepen van functies

Begin met "def" en volg het op met de functie naam en "():" om een functies te creëren. Net als bij een if uitdrukking moet de code zijn ingesprongen. De functie wordt aangeroepen met de functienaam gevolgd door "()".

Parameters stellen en argumenten

Functies maken het mogelijk om op basis van de inputs verschillende outputs te geven inputs. Hiervoor worden parameters en argumenten gebruikt:

def wash_car(amount_paid):
    if amount_paid == 12:
        print("Wash with tri-color foam")
        print("Rinse twice")
        print("Dry with large blow dryer")
    if amount_paid == 6:
        print("Wash with white foam")
        print("Rinse once")
        print("Air dry") 
wash_car(12)

In het bovenstaande voorbeeld wordt de variabel amount_paid gedefinieerd tussen de haakjes van de functie. Deze variabel wordt een parameter genoemd en stelt de functie instaat om veranderingen te maken op basis van de input. De waarde die hoort bij deze variabel, wordt een argument genoemd. Een argument wordt gegeven door een waarde tussen de haakjes te plaatsen tijdens het aanroepen.

Waardes terugkeren van functies

Functies kunnen worden gebruikt om waardes terug te geven. Een return uitdrukking wordt gebruikt om te definiëren wat deze waarde moet zijn, Een return uitdrukking bestaat uit het return sleutelwoord en de waarde die de functie moet teruggeven.

Functies in verschillende talen
