Learning Python (2021)

Van start gaan

Python installeren op Windows

Op een Windows-computer kan worden gecontroleerd of Python correct is geïnstalleerd door in een terminalvenster het commando py --version uit te voeren. Indien het resultaat niet de melding "Python 3.x" (minimaal 3.9) weergeeft of er een foutmelding verschijnt, is installatie van de juiste versie noodzakelijk.

Ga naar de officiële Python-website via python.org en klik op de knop Downloads, waarbij automatisch de aanbevolen versie voor Windows wordt geselecteerd (minimaal Python 3.9). Download vervolgens de installer en voer deze uit. De installatie duurt doorgaans enkele minuten.

Open een nieuw terminalvenster na de installatie en voert men wederom py --version uit om te bevestigen dat Python nu correct is geïnstalleerd.

Python installeren op de MAC

Veel macOS- en Linux-systemen komen met Python 2 én Python 3. Controleer in de terminal met python3 --version of de geïnstalleerde Python-versie ten minste 3.9 is. Zo niet, download en installeer Python 3 vanaf de officiële website en test vervolgens opnieuw in de terminal.

Op een Mac (of Linux) staat vaak standaard een oudere versie van Python geïnstalleerd. Om te controleren of de geïnstalleerde versie voldoet wordt de terminal geopend en wordt het python3 --version commando uitgevoerd. Indien het resultaat niet "Python 3.x" (minimaal 3.9) toont of er een foutmelding verschijnt, moet een nieuwere versie worden geïnstalleerd.

Ga naar python.org en klik op Downloads en selecteer de installer voor Mac (de pagina past zich meestal automatisch aan op je besturingssysteem). Download en voer de installer uit. Dit proces duurt doorgaans enkele minuten. Het python3 --version commando wordt opnieuw uitgevoerd om te verifiëren dat nu "Python 3.9 (...)" of hoger wordt gerapporteerd.

Een editor of IDE kiezen

Hoewel elke teksteditor kan worden gebruikt, wordt Visual Studio Code (VS Code) aanbevolen vanwege de uitgebreide ondersteuning voor Python. Daarnaast is VS Code gratis, draait het op Windows, Mac en Linux, en beschikt het over uitbreidingen die debugging en linting gemakkelijk maken.

Ga naar code.visualstudio.com en download de versie voor het eigen besturingssysteem. Voer de installer uit en start VS Code na afronding van de installatie.

Om de Python-extensie te installeren wordt er op het "Extentions" tab geklikt, op "Python" gezocht en vervolgens voor de Microsoft extensie gekozen. Deze extensie installeert automatisch ook de Pylance-taalserver.

Open de map met oefenbestanden als een project in VS Code (via Open Folder of drag-&-drop). Onderin de statusbalk verschijnt de melding met de gekozen Python-versie. Klik hierop om een pop-up te openen en selecteer de juiste Python-interpreter (bijv. Python 3.9).

Indien de statusbalk-melding niet zichtbaar is, gebruik dan de Command Palette (Ctrl+Shift+P op Windows, Cmd+Shift+P op Mac) en voer "Python: Select Interpreter" uit. Kies vervolgens de gewenste interpreter.

Open de instellingen (Ctrl+, op Windows, Cmd+, op Mac) en zoek in het zoekvak naar executeInFileDir. Vink deze voorkeur aan. Dit zorgt ervoor dat VS Code bij het uitvoeren van scripts standaard naar bestanden zoekt in dezelfde map als het bronbestand.

Voorbeelden van hoe Python wordt uitgevoerd

Bestanden kunnen ook buiten een IDE worden uitgevoerd, door via de terminal (op Windows PowerShell of Opdrachtprompt; op Mac/Linux Terminal) te werk te gaan. Het cd-commando wordt gebruikt om naar de locatie van de map te navigeren. Als de map op het bureaublad staat ziet dit er bijvoorbeeld als volgt uit: cd %USERPROFILE%\Desktop\ExerciseFiles

De inhoud van de map kan met ls (Mac/Linux) of dir (Windows) worden gecontroleerd. Vervolgens kan naar de gewenste submap worden genavigeerd door bijvoorbeeld het volgende de toen: cd Chapter2

Een bestand kan worden uitgevoerd door py of python te typen gevolgd door de bestandsnaam (zonder extensie als .py opgenomen is in de instellingen). Bijvoorbeeld: python helloworld_finished.py.

Voor de Mac wordt altijd gebruik gemaakt van het python3 commando om de Python 3-interpreter aan te roepen: python3 helloworld_finished.py

Python basiskennis

Hello World bouwen

In de terminal wordt python gestart met het py of python commando op Windows en met het python3 commando op macOS en Linux. Code kan direct in de terminal worden geschreven en uitgevoerd, bijvoorbeeld:

>>> print("Hello world")
Hello world

Met exit() of quit() wordt de geïnterpreteerde modus afgesloten en keert de gebruiker terug.

Met behulp van de Python-extensie in VS Code kan het bestand direct in de ingebouwde terminal worden uitgevoerd via "Run Python File in Terminal". Vervolgens wordt de code in een functie opgedeeld om de logica te groeperen en herbruikbaarheid te versterken. Dit gebeurt door het aanmaken van een functie main() en het gebruik van een if __name__ == "__main__":-blokkering.

Hierdoor wordt onderscheid gemaakt tussen het uitvoeren van het bestand als zelfstandig script en het importeren als module in een ander script. De meest relevante syntactische elementen zijn hieronder weergeven:

Syntax Uitleg Voorbeeld
def functieNaam(): Definieert een functie. Alle daaropvolgende regels met identieke inspringing behoren tot die functie. 
def main():
    print("Hello World")
if __name__ == "__main__": Controleert of het script als hoofdprogramma wordt uitgevoerd. De code in dit blok wordt alleen uitgevoerd bij direct starten, niet bij importeren. 
if __name__ == "__main__":
    main()
print() Roept de ingebouwde Python-functie aan om tekst of andere waarden naar de standaarduitvoer te sturen. 
print("Hello World")
input() Roept de ingebouwde Python-functie aan om een invoerprompt te tonen en de invoer als string terug te geven. 
name = input("What is your name? ")
Inspringing (spaties of tabs) Geeft in Python de hiërarchie en codeblokken aan. Alle regels binnen een blok moeten exact dezelfde inspringing hebben. 
    print("")  # Vier spaties
exit() of quit() Sluit de interactieve modus af en keert terug naar de opdrachtregel. 
exit()

Variabelen en expressies

In Python zijn er vijf datatypen die gebruikt worden om informatie op te slaan en te manipuleren:

Datatype Uitleg Voorbeeld
Numbers Gehele getallen (int) en kommagetallen (float) 
42, 3.14
Strings Reeksen van teksttekens 
"Hello world"
Booleans Waarheidswaarden 
True, False
Sequenties Verzamelingen zoals lijsten (list) en tuples (tuple) 
[1, 2, 3], ("a", "b", "c")
Dictionaries Sleutel/waarde-paren 
{"een": 1, "twee": 2}

De toekenning van datatypes is dynamisch in Python, wat inhoudt dat variabelen opnieuw kunnen worden toegekend met een ander gegevenstype, bijvoorbeeld:

myint = 5
myint = "abc"

Sequentiële datatypen zoals lists en tuples maken gebruik van indexering en slicing:

Syntaxis Uitleg Voorbeeld
mylist[2] Toegang tot het derde element (index start bij 0) 
[1, 2, 3, 4][2] → 3
mylist[1:5] Slicing van index 1 t/m 4 
[2, 3, 4, 5]
mylist[1:5:2] Slicing met een stap van 2 
[2, 4]
mylist[::-1] Keert de volgorde van de lijst om 
[5, 4, 3, 2, 1]

Dictionaries gebruiken sleutels in plaats van indexen, bijvoorbeeld:

mydict = {"een": 1, "twee": 2}
print(mydict["een"])  # → 1

De sleutels moeten uniek zijn (waarden niet) en worden tussen aanhalingstekens genoteerd als het strings zijn.

Python is sterk getypeerd, wat inhoudt dat verschillende typen niet zomaar gecombineerd kunnen worden. Het volgende is dus niet mogelijk binnen Python: print("waarde: " + 123). In plaats daarvan moet dit worden gebruikt: print("waarde: " + str(123)).

Variabelen kunnen zowel globaal als lokaal zijn gedefinieerd:

mystr = "globale waarde"

def somefunction():
    mystr = "lokale waarde"
    print(mystr)

somefunction()
print(mystr)

VS

mystr = "globale waarde"

def somefunction():
    global mystr
    mystr = "aangepaste globale waarde"
    print(mystr)

somefunction()
print(mystr)

Een variabele binnen een functie overschrijft niet automatisch een globale variabele met dezelfde naam.

Ten slot is het ook mogelijk om bestaande variabelen te verwijderen:

Statement Uitleg Voorbeeld
del Verwijderd een variabele 
del mystr

Python functies

Functies in Python bieden een manier om code te organiseren in afzonderlijke, herbruikbare blokken. Elke functie wordt gedefinieerd met het sleutelwoord def, gevolgd door een functienaam, optionele parameters tussen ronde haakjes en een dubbele punt. Daarna volgt de functiebody, afgebakend door inspringing.

Een functie zonder argumenten wordt gedefinieerd met def functienaam():. De statements binnen de functie worden ingesprongen ten opzichte van de regel met def.

Sleutelwoord Uitleg Voorbeeld
def Definieert een functie. 
def func1():
    print("I am a function")

Functies kunnen gedefinieerde parameters accepteren; deze worden in volgorde toegewezen aan de doorgegeven argumenten. Met het sleutelwoord return kan een waarde worden teruggegeven.

Sleutelwoord Uitleg Voorbeeld
return Geeft een waarde terug. 
def func2(arg1, arg2):
    print(arg1, arg2)

def cube(x):
    return x * x * x

Parameters kunnen standaardwaarden krijgen met de notatie param=standaardwaarde. Wordt bij aanroep geen waarde voor dat argument opgegeven, dan wordt de standaardwaarde gebruikt. Bij aanroep via naam (keyword-argumenten) hoeven de argumenten niet in volgorde te staan.

Met een sterretje (*) vóór een parameternaam kan een functie een variabel aantal niet-naamgegeven argumenten ontvangen. Deze verzamelt Python in een tuple.

def multi_add(*args):
    result = 0
    for x in args:
        result += x
    return result

De parameter met *args moet altijd als laatste staan.

Voorwaardelijke structuren

Voorwaardelijke logica speelt een cruciale rol in vrijwel elk programma: op basis van één of meer voorwaarden wordt bepaald welke code uitgevoerd wordt. In Python bestaan hiervoor verschillende mogelijkheden: de klassieke if-elif-else-blokken, de beknopte voorwaardelijke expressie (ternary operator) en sinds Python 3.10 de match case-constructor.

Met een if-elif-else-structuur kan een programma voorwaarden achtereenvolgens evalueren en afhankelijk daarvan een bepaalde tak uitvoeren:

x = 10
y = 100

if x < y:
    result = "x is less than y"
else:
    result = "x is greater than y"
print(result)

Wanneer niet alleen de twee basistakken nodig zijn, maar ook een derde optie (bijvoorbeeld "gelijk aan"), worden er één of meer elif-takkken toegevoegd:

x = 100
y = 100

if x < y:
    result = "x is less than y"
elif x == y:
    result = "x is the same as y"
else:
    result = "x is greater than y"

print(result)

Voor simpele "als-dan-anders"-logica biedt Python de compacte syntax:

result = "x is less than y" if x < y else "x is greater or equal to y"
print(result)

Het voordeel hiervan is dat er weinig regels code nodig zijn voor een eenvoudige voorwaarde. Deze syntax is echter minder geschikt bij complexere of geneste voorwaarden.

Net als bij functies en lussen moet de code binnen if, elif, else en case steeds exact evenveel spaties (of tabs) ingesprongen zijn. Aangezien Python de voorwaarden van boven naar beneden controleert, dient de meest specifieke voorwaarde ook het eerst te komen wanneer een overlappende range mogelijk is en komt Else altijd als laatste.

Vanaf Python 3.10 is de match case-constructor beschikbaar. Deze vervangt in veel gevallen opeenvolgende if-elif-takken, vooral wanneer één variabele tegen meerdere vaste waarden moet worden vergeleken:

value = "three"

match value:
    case "one":
        result = 1
    case "two":
        result = 2
    case "three" | "four":
        result = (3, 4)
    case _:
        result = -1

print(result)
Structuur Toepassing Kenmerken
if ...: ... elif ...: ... else ...: Klassieke vertakkingslogica - Flexibel, meerdere condities in volgorde.
- Elke elif-tak wordt alleen getest als alle voorgaande condities False waren.
<expr1> if <cond> else <expr2> Eéngeregelige voorwaarde - Compact voor simpele gevallen.
- Niet leesbaar bij ingewikkelde logica.
match ...: case ...: ... case ...: ... case _: ... Veelvoudige vergelijkingen tegen één variabele - Zeer leesbaar bij vele vaste waardecontroles.
- Vereist Python 3.10 of hoger.

Loops

Python biedt twee soorten loops aan om herhaaldelijk een blok code uit te voeren: de while- en de for-loop:

Looptype Uitleg Voorbeeld
while <conditie>: Voert de lusbody steeds opnieuw uit zolang de voorwaarde True is. 
x = 0

while x < 5:
    print(x)
    x = x + 1
for <variabele> in <iterable>: Itereert over alle items van een iterabel. Geen expliciete index nodig. 
for x in range(5, 10):
    print(x)

In Python is het mogelijk om loops te onderbreken of om een de rest van de lusbody over te slaan. Dit staat hieronder weergeven:

Constructor Uitleg Voorbeeld
break Stopt met de lus, ongeacht de oorspronkelijke conditie. 
for x in range(5, 10):
    if x == 7:
        break
    print(x)
continue Slaat de rest van de lusbody over en ga direct naar de volgende iteratie. 
for x in range(5, 10):
    if x % 2 == 0:
        continue
    print(x)

Hoewel standaard geen teller gebruikt wordt in een Python-for-lus, kan enumerate() worden gecombineerd om zowel index als waarde op te halen:

Constructor Uitleg Voorbeeld
for i, item in enumerate(iterable): Haalt zowel index als waarde op tijdens de iteratie. 
days = ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday"]

for i, d in enumerate(days):
    print(i, d)

Klassen

In Python is object-georiënteerd programmeren de standaard. Klassen bieden een manier om gerelateerde gegevens en functies te groeperen tot herbruikbare, onderhoudbare modules. Een klasse wordt aangemaakt met het sleutelwoord class, gevolgd door de klassennaam en een mogelijke superklasse tussen haakjes. Daarna volgt een dubbele punt en de ingesprongen klassebody.

Sleutelwoord Uitleg Voorbeeld
class Definieert een nieuwe klasse. Tussen haakjes kan een (super)klasse staan voor overerving. 
class Vehicle:
    def __init__(self, body_style):
        self.body_style = body_style

Binnen een kunnen de volgende constructor's worden gebruikt:

Constructor Uitleg Voorbeeld
__init__ Wordt automatisch aangeroepen bij het maken van een object. 
def __init__(self, body_style):
self Verwijst naar de instantie zelf (vergelijkbaar met this in andere talen). 
self.body_style = body_style

Wanneer meerdere klassen gemeenschappelijke eigenschappen delen, kan een nieuwe klasse erven van een bestaande (super)klasse:

Sleutelwoord Uitleg Voorbeeld
super() Roept de constructor of methode van de superklasse aan. 
super().__init__(body_style)

Dit gebeurt door in de definitie van de subklasse de naam van de superklasse tussen haakjes te plaatsen:

class Motorcycle(Vehicle):
    def __init__(self, body_style, engine_type, has_sidecar):
        super().__init__(body_style)
        if has_sidecar:
            self.wheels = 3
        else:
            self.wheels = 2
        self.doors = 0
        self.engine_type = engine_type

Importeren en modulen gebruiken

Python beschikt over een uitgebreide verzameling aan libraries die voorgebouwde functionaliteiten bevatten. Door deze modules te importeren, hoeven bestaande complexe functies niet opnieuw geschreven te worden. Het importeren van een module gaat als volgt:

Sleutelwoord Uitleg Voorbeeld
import Laadt een module in en maakt het beschikbaar in het huidige script 
import math

Vervolgens kan er met <moduleNaam>.<functie> of <moduleNaam>.<constante> gebruik worden gemaakt van alle publieke functies en waarden uit die module, bijvoorbeeld:

import math
print("De vierkantswortel van 16 is", math.sqrt(16))

De bovenstaande voorbeelden maakten beiden gebruik van de math-module. Hieronder worden nog een aantal modules toegelicht:

Module Uitleg
math Wiskundige functies en constante waarden, zoals vierkantswortel, trigonometrie, logaritmen en de wiskundige constante π.
datetime & pathlib Data- en tijdmanipulaties zoals datums optellen, formaten omzetten, etc.
os Werken met bestandensystemen zoals bestanden lezen, mappen aanmaken, paden bewerken.
json & xml Inkomsten van en schrijven naar Internet-gebaseerde dataformaten.

Uitzonderingen gebruiken

Tijdens het uitvoeren van code kunnen diverse fouttypes zich voortdoen die tot onverwachte situaties zullen leiden:

Fouttype Uitleg Voorbeeld
ZeroDivisionError Treedt op wanneer er door 0 wordt gedeeld. 
10 / 0
ValueError Vindt plaats wanneer een conversie van string naar getal mislukt. 
int("abc")
FileNotFoundError Treedt op als geprobeerd wordt een niet-bestaand bestand te openen. 
open("onbestaand.txt")
KeyError Komt voor bij het opvragen van een niet-bestaande sleutel in een dict. 
mijn_dict["nietbestaan"]
IndexError Treedt op wanneer een niet-bestaande index in een lijst of tuple wordt benaderd. 
[1,2,3][5]

Wanneer een onvoorziene situatie optreedt, is het beter om deze op gecontroleerde wijze af te handelen in plaats van het programma abrupt te laten crashen. Python heeft de try-except-finally-constructor om dit leesbaar en gestructureerd te doen:

try:
    # Code die mogelijk een fout veroorzaakt
    x = 10 / 0
except:
    # Foutafhandeling voor álle uitzonderingen
    print("Er is iets misgegaan.")
finally:
    # Dit wordt altijd uitgevoerd, ongeacht of er een fout optrad
    print("Dit deel draait altijd.")
Constructor Uitleg
try: Omringt code die wél of niet faalt. Indien er een uitzondering optreedt, springt Python naar het corresponderende except-blok.
except <FoutType> as <variabele>: Vangt specifiek fouttype <FoutType> op. <variabele> bevat het bijbehorende uitzonderingobject (Exception) met details over de fout.
except: Vangt alle niet eerder afgehandelde fouttypes op (een generieke vangnet-oplossing).
finally: De code in dit blok wordt áltijd uitgevoerd, ook als het try- of except-blok abrupt eindigt of return-instructies bevat. Perfect voor cleanup.
else: (optioneel) Kan worden toegevoegd na alle except-blokken. De code in else loopt alleen als geen enkele uitzondering is opgetreden in het try-blok.

Om verschillende fouten apart te behandelen, kan kunnen er meerdere except-blokken onder elkaar worden gezet, elk met een specifiek fouttype. Daarbij kan de uitzonderingsinformatie worden opgevangen in een lokale variabele (bijv. except ValueError as e).

try:
    answer = input("Waarvan wil je 10 delen? ")
    number = int(answer)
    result = 10 / number
    print("Uitkomst:", result)

except ZeroDivisionError as e:
    print("Je kunt niet door nul delen.")

except ValueError as e:
    print("Geen geldig getal ingevoerd:", e)

finally:
    print("Dit deel wordt altijd uitgevoerd, ongeacht of er een fout was.")

Met bestanden werken

Bestanden lezen en schrijven

Python bevat ingebouwde functies voor het werken met bestanden en mappen. Hiermee kunnen bestanden worden aangemaakt, beschreven, gelezen en uitgebreid. Het openen van een bestand gaat aan de hand van de onderstaande functie:

Functie Uitleg Voorbeeld
open Opent een bestand 
myfile = open("textfile.txt", "w+")

De open-functie accepteert twee argumenten: De bestandsnaam en de modus waarin het bestand geopend moet worden:

Modus Uitleg Voorbeeld
"w" Schrijft naar bestand, bestaande inhoud wordt overschreven 
myfile = open("textfile.txt", "w")
"w+" Schrijft en creëert het bestand indien het niet bestaat 
myfile = open("textfile.txt", "w+")

De open-functie kan met diverse methodes worden uitgebreid, om in een bestand te schrijven, om inhoud toe te voegen of om een bestand te lezen:

Methode Uitleg Voorbeeld
.write() Schrijft een bestand 
myfile = open("textfile.txt", "w+")
for i in range(10):
    myfile.write("Dit is een tekstregel.\n")
myfile.close()
read() Leest de gehele inhoud van een bestand in een keer. 
myfile = open("textfile.txt", "r")
if myfile.mode == "r":
    contents = myfile.read()
    print(contents)
readlines() Leest de gehele inhoud van een bestand regel per regel. 
myfile = open("textfile.txt", "r")
lines = myfile.readlines()
for line in lines:
    print(line)

Bestanden die met de open()-functie zijn geopend worden altijd afgesloten met .close(). Bij de read()-methode is .close() optioneel, maar wordt vaak toch gebruikt voor netheid en geheugenbeheer.

Om regels toe te voegen aan een reeds bestaand bestand, wordt de a-modus gebruikt (append). De bestaande inhoud blijft behouden.

Modus Uitleg Voorbeeld
"a" Voegt een regel toe aan een bestaand bestand. 
myfile = open("textfile.txt", "a")
for i in range(10):
    myfile.write("Dit is een nieuwe tekstregel.\n")
myfile.close()

Werken met OS path utilities

Python biedt modules en functies om bestanden en mappen te onderzoeken, paden te verwerken en datum/tijdinformatie van bestanden op te halen. In deze sectie worden de belangrijkste stappen en functies samengevat.

Module Toepassing
os Bevat algemene functies voor interactie met het besturingssysteem.
os.path Submodule van os voor padgerelateerde bewerkingen.
time Voor conversies tussen tijdstempel en leesbare datum/tijd.
datetime Voor geavanceerde datum/tijdmanipulaties.

Met os.name kan worden bepaald welk type besturingssysteem wordt gebruikt. Op Windows geeft dit 'nt' weer (intern de officiële naam van Windows) en op Linux/macOS geeft dit meestal 'posix'.

Met functies uit os.path kan worden getest of een item bestaat en of het een bestand of map is. Ook is het mogelijk om het volledige pad (incl. bestandsnaam) te achterhalen en vervolgens pad en bestandsnaam te splitsen:

Functie Uitleg Voorbeeld
exists() Controleert of het pad (bestand of map) bestaat 
item = "textfile.txt"
print("Bestaat:", path.exists(item))
isfile() Geeft True als het pad verwijst naar een bestaand regulier bestand 
item = "textfile.txt"     
print("Is bestand:", path.isfile(item))
isdir() Geeft True als het pad verwijst naar een bestaande map 
item = "textfile.txt"
print("Is map:", path.isdir(item))
realpath() Geeft het kanonieke (absolute) pad terug. 
full_path = path.realpath("textfile.txt")
split() Splitst een pad in een tuple (pad_zonder_bestandsnaam, bestandsnaam). 
full_path = path.realpath("textfile.txt")
dir_part, file_part = path.split(full_path)
print("Volledig pad:", full_path)
print("Pad zonder bestandsnaam:", dir_part)
print("Alleen bestandsnaam:", file_part)

Met modules is het tevens mogelijk om de wijzigingstijd van een bestand op te halen Met path.getmtime() wordt de wijzigingstijd (in seconden sinds de UNIX-epoch) opgehaald:

Functie Uitleg Voorbeeld
getmtime() Geeft wijzigingstijd als UNIX-tijd (float, seconden sinds 1970-01-01). 
mtime_ts = path.getmtime("textfile.txt")
ctime() Zet UNIX-tijdstempel om naar een leesbare datum/tijdstring. 
print("Wijzigingstijd (timestamp):", mtime_ts)
leestijd = time.ctime(mtime_ts)
print("Leesbare wijzigingstijd:", leestijd)

Met datetime.fromtimestamp() wordt de timestamp omgezet naar een datetime-object:

Methode Uitleg Voorbeeld
fromtimestamp() Converteert een UNIX-tijdstempel naar een datetime-object met datum en tijd. 
dt_obj = datetime.datetime.fromtimestamp(path.getmtime("textfile.txt"))
print("Wijzigingstijd als datetime:", dt_obj)

Bepalen hoe lang geleden een bestand is gewijzigd, is mogelijk door de huidige tijd te vergelijken met de wijzigingstijd ontstaat een timedelta-object:

now = datetime.datetime.now()
modified = datetime.datetime.fromtimestamp(path.getmtime("textfile.txt"))
delta = now - modified

def print_results(json_data):

Soms is het wenselijk om alleen gebeurtenissen met een bepaalde magnitude te tonen. Elke gebeurtenis heeft een sleutel "mag". Hieronder wordt een lijst afgedrukt van alle aardbevingen met magnitude 4.0 of hoger:

def print_results(json_data):
    the_json = json.loads(json_data)
        
    print("Aardbevingen met magnitude ≥ 4.0:")
    print("-------------------------------")
    
    for feature in the_json["features"]:
        mag = feature["properties"]["mag"]
        if mag is not None and mag >= 4.0:
            place = feature["properties"]["place"]
            print(f"{mag} – {place}")
    print("\n---\n")

De JSON-gegevens bevatten in "properties" vaak een "felt"-eigenschap: het aantal meldingen van mensen die de aardbeving hebben gevoeld. Door alleen die gebeurtenissen te selecteren waarbij ten minste één persoon melding heeft gemaakt, ontstaat een lijst van 'gevoelde' bevingen:

def print_results(json_data):
    the_json = json.loads(json_data)
    
    print("Aardbevingen die zijn gevoeld:")
    print("-------------------------------")
    
    for feature in the_json["features"]:
        felt_reports = feature["properties"]["felt"]
        if felt_reports is not None and felt_reports > 0:
            place = feature["properties"]["place"]
            mag = feature["properties"]["mag"]
            print(f"{place} – mag: {mag}, felt: {felt_reports} keer")
